Project Idea

Overview

A companion app to Pixels Dice, found here, which are a set of electronic dice to be used in tabletop role playing games (TTRPG). TTRPGs will often be based around a 'character sheet' which will contain information on the player's character, such as their particular modifiers. The companion app will be based around these character sheets, allowing the Pixels Dice to interact with them and help with the mathematical aspects of TTRPGs. The app will also provide a built-in character sheet creator, and will have some functionality to help people new to TTRPGs. It will also have some functionality to integrate with other services that are already available.

Motivation

A website used for playing table-top role playing games, Roll20, reported 8 million users in 2020. While it is a Niche hobby, there are many people playing TTRPG's and there have been many pieces of technology that have been developed because of it, including virtual tabletop (VTT) programs and websites such as Roll20. These technologies make the games easier to play, as they aid with the visualisation and mathematic aspects of the games, resulting in a 'modern' style of playing. Despite the creation of these digital tools, many people prefer to play in a more traditional pen and paper only style, as they feel that it harms the experience of the games. With the my project, I will be attempting to bridge the gap between the traditional pen and paper style of play, to the modern style of play.

Description

The target audience I have in mind for this app are people that are new to these types of games, and people that have trouble with the mathematical aspects of the game. The app will mostly support Dungeons and Dragons 5th Edition, as it is the most played game in the TTRPPG genre. The app would also be great for more math heavy TTRPGs such as Pathfinder. The main feature of the app is being able to detect the numbers rolled on the Pixels Dice, and automatically add or subtract any relevant modifiers. In order to achieve this, the app and its main features will be primarily focused on interactive character sheets which holds the relevant data to interact with the Pixels Dice. Because of this, there will be many features built in dedicated to making character sheets. To support other TTRPGs, there would need to be additional support for them, as each TTRPG uses their own unique sheets. It will also have features that support importing already made character sheets. This includes online character sheet makers like DNDBeyond, or any official character sheet PDFs. Because one of the main focuses of the app will be to help those new to the hobby, the built-in character sheet maker will have an option to enable a tutorial mode, which guides the user through creating a character sheet. The user would interact with the character sheet, denoting which action and modifier they are using. Once a modifier is defined, the user would then roll the appropriate Pixels Dice. The Pixels Dice will detect the number rolled, send it to the app, which will perform the calculation and give the user the result. There will also be support for manually entering modifiers and rolls, as TTRPGs can be very spontaneous and don't have hard set rules to follow. Additionally, there will be features that allow users to store any values in case they are needed, or if multiple rolls are required (such as rolling twice and taking the higher value). TTRPGs more often than not use multiple different sided dice, and so to help new players, when a modifier or action is picked, the corresponding dice that the user needs to roll will light up. This will help newer players familiarise themselves with which die is which. TTRPGs are inherently collaborative, so there will also be options to connect with other players, allowing them to see what each other rolls, if they so choose. One criticism that VTTs get is that the tactile feeling of rolling dice is removed, as everything is digital, and often done by the click of a button, so another feature of the app will be to send data to VTTs allowing them to be used for online play. While the Pixels dice already has official support for implementing integrations to popular services such as Roll20, this app will need custom modules for these servicesâ€™ proper integration. But these features will come later as they are not the main focus of the project.

Tools and technology

The Pixels Dice will be the main piece of hardware and software that is needed. Fortunately, the creator of the Pixels Dice, Jean Simonet, is committed to keeping the API Open Source on a GitHub repository. Not only that, but the project is also Open Hardware, with all the layouts, schematics, and a list of components available on Hackaday. The app will be easily developed with the Open Source available (as well as a discord server filled with other like-minded people), but assembling the hardware requires additional equipment, such as a soldering iron, pressure pot, and other dice-making equipment.

Skills required

The software for Pixels Dice is written primarily in C, C++, and C#. The firmware is built with Nordic's nRF5 SDK. This is what is required to develop apps that work with the Dice. Most VTTs also use HTML, CSS, JavaScript, and JSON, when developing custom modules, such as one that allows for integration with this project. The Pixels Dice themselves can be bought through the Kickstarter. Developers can make their own Dice, as there are schematics and component lists on the Hackaday page. These boards require assembly, from soldering batteries to folding and gluing. Once the board has been assembled it follows dice making procedure; the board is encased in resin, put in a pressure pot, and then sanded down. Overall, developing programs for the dice requires experience in coding languages, which are fairly easy to learn. Creating the hardware may prove to be more difficult, and cost intensive.

Outcome

If the project is successful, I would expect that the line between the traditional pen and paper style of play and the modern style of play gets blended. I would also expect that newer players would have an easier time learning and playing TTRPGs, due to the beginner friendly features. Older and more experienced players can also benefit from its functionality as it helps with the mathematical aspects of the games, which gets more complicated the longer you play. Despite this, the cost of Pixels Dice is very high, compared to other RPG dice that is available at local game stores.
